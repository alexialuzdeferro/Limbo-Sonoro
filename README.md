## 🌑 Sobre

Sejam muito bem-vindos ao **Limbo Sonoro — Anatomia do Abismo**, um projeto de tom visceral conduzido por **Alexia Melusine**.

Este repositório é uma das câmaras desse abismo — onde textos escritos há **mais de três décadas** encontram forma digital. Cápsulas de memória de uma época que não volta, mas que insiste em ter som, peso, respiração.

> *"Onde o som é ponte para lugares onde nunca voltamos."*
